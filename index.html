Your Firebase project is blocking access. You must update your Security Rules in the Firebase Console to allow read/write operations for both Database and File Uploads.

1. Firestore Database Rules:

rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /{document=**} {
      allow read, write: if request.auth != null;
    }
  }
}

2. Storage Rules (For Files/Images):

rules_version = '2';
service firebase.storage {
  match /b/{bucket}/o {
    match /{allPaths=**} {
      allow read, write: if request.auth != null;
    }
  }
}
